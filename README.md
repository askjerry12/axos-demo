# Axos Bank — Builder.io Demo Replica

A pixel-perfect static replica of [axosbank.com](https://www.axosbank.com) built for Builder.io sales demos.

**Live Preview:** [axos-demo.vercel.app/axos](https://axos-demo.vercel.app/axos)

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Setup

```bash
# Clone the repo
git clone https://github.com/askjerry12/axos-demo.git
cd axos-demo

# Install dependencies
npm install
```

## Development

```bash
# Start the dev server
npm run dev
```

Open [http://localhost:3000/axos](http://localhost:3000/axos) in your browser.

## Production Build

```bash
npm run build
npm start
```

## Deploy to Vercel

```bash
vercel --prod
```

## Notes

- All content is static/mock — no external API calls
- Images are downloaded from the real Axos Bank site for visual accuracy
- This is a demo replica for Builder.io sales presentations only
